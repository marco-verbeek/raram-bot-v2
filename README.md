## Installation

Install all dependencies using

```
npm install
```

and create a .env file with following variables:

|     VARIABLE     |        DESCRIPTION         |
| :--------------: | :------------------------: |
|  DISCORD_TOKEN   | Your Discord Bot API token |
| DISCORD_OWNER_ID |    Your Discord User ID    |
